# D9ScrollRect

유니티 스크롤 대체용으로 만든 ScrollRect UI 입니다

### 개선점
* 짧은 터치도 인식 가능 (튕기기 스크롤 대응)
* 이중 Scroll Rect 가능
* 움직인 거리 및 시간에 알맞게 가속도 적용
